3. Test it:

   - Type: 1737025200

   - Press ⌘⇧T

🎯 SUPPORTED FORMATS:

INPUT:                          OUTPUT:
• Unix seconds                  → All other formats
• Unix milliseconds             → All other formats
• Unix microseconds             → All other formats
• Unix nanoseconds              → All other formats
• ISO 8601 dates                → All other formats
• Human-readable dates          → All other formats

Examples:
  1737025200          → January 16, 2025
  1737025200000       → 1737025200 seconds
  2025-01-16T10:00:00Z → 1737025200
